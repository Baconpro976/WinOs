# WinOs

Custom Debian 12 Live ISO with a Windows 11-like interface.

## Features

- Debian 12 Bookworm
- XFCE4 Desktop
- Windows 11 Theme
- PipeWire Audio
- Non-Free Firmware
- Firefox
- GParted
- GitHub Actions ISO Builder

## Build Instructions

1. Fork or upload this repository to GitHub.

2. Go to:

Actions → Build WinOs ISO

3. Click:

Run workflow

4. Wait 30–90 minutes.

5. Download the ISO from:
   - Actions Artifacts
   - Releases section

ISO Name:

WinOs-v1.iso
